# Title
Submerged Survival

## Repository
https://github.com/AveryV11/final-project

## Description
This project is a simple interactive arcade-style game where the player (submarine) must avoid falling objects (sea monsters). It demonstrates digital arts and media through visual design, animation, and user interaction.

## Features
- Player Movement
  The player moves left and right using keyboard controls

- Falling Objects 
  Objects generate randomly at the top of the screen and fall downward, increasing in speed over time.

- Collision Detection 
  The game detects when the player is hit by an object and ends the game.

- Score System
  The player's score increases based on how long they survive.

- Game Over and Restart
  A game over screen appears with an option to restart the game.

- Visual Assets
  The player and objects are displayed using personally drawn images instead of simple shapes to improve visual quality.

## Challenges
- Execute collision detection between objects.
- Learning how to display images instead of a simple shape.
- Managing different game states (playing, game over).

## Outcomes
Ideal Outcome:
- A skillful game with personally drawn assets(Background, Submarine, and Sea Monsters), inceasing difficulty, and a streamlined user interface.

Minimal Viable Outcome:
- A working game where the player can move, avoid objects, track score, and restart after losing.
